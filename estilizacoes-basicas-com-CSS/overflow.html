A propriedade overflow

A propriedade overflow especifica se o conteúdo deve ser cortado ou adicionado barras de rolagem quando o conteúdo de um elemento é grande demais para caber em uma área especificada.

overflow: scroll: O conteúdo é cortado, mas uma barra de rolagem é adicionada para visualiza-lo
