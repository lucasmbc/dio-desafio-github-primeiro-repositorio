Seletores CSS

Bem-vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por Tipo: Busca elemento por uma tag HTML
  • Seletor por ID (#): Busca elementos através do atributo "id"
  • Seletor por Classe (.): Busca elementos através do atributo "class"
  • Seletor por Universal (*): Seleciona todos os elementos HTML
  • Seletor de Atributo ([atrib]): Seleciona elementos que possuem um atributo específico em nosso documento HTML e consequimos também buscar atributos com um valor específico
